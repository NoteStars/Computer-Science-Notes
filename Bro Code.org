** On the spectrum, there are high-level languages or low-level languages (for computers).
*** Computers only understand binary code. However, for humans to understand this language, we program using source code which is understandable by humans.
*** Source Code (.java file) -> Compile -> Object Code (Machine Code)

*** There is a problem when we compile on different machines. When we compile, how will platforms like Mac and Windows compile?
**** Java has a solution for this and it is called a JVM (Java Virtual Machine).
**** This is also known as a Byte code and ends with a .class file extension.
**** JVM -> Object Code
